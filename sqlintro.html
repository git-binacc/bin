What is SQL?

Structured Query Language (SQL) is the standard programming language used to manage, manipulate and retrieve data stored in relational databases. It allows you to communicate with databases using simple commands like CREATE, RETRIVE, UPDATE and DELETE to handle large amounts of data efficiently.

To understand SQL, we have to know about the basics terms like data & database. Difference between DBMS and RDBMS. How they these are related to each other ?

v1.1-dbms

DBMS - Manager

DataBase Management System shortly as DBMS, which is used to maintain the database, like a store manager who stocks shelves and helps you to find items easily & rapidly.

A software that manages data in a structured way, often using files ( or ) hierarchical models. To communicate with DBMS software, we use QL ( Query Language ).

v1.1-rdbms

RDBMS - Organised Layout

Relational Database Management System, a type of DBMS that organises data in tables ( relations ) with rows and columns. It enforces relationships using primary keys & foreign keys.

-> To communicate with RDBMS, we use SQL ( Structured Query Language ).

The main reason to introduce the RDBMS is, to overcome the storage issues associated with earlier database systems. By providing a structured approach to data management, RDBMS ensures data integrity, reduces redundancy and facilitates efficient querying and updating of information.

Features in DBMS & RDBMS

There are 2 main features:

Security : Individual password ( users have their own password to access the database )

Authorisation : Give access to specific users based on their roles with permissions.
